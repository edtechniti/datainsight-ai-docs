# Line Charts

Line charts are essential for visualizing trends and changes over time. They connect data points with lines, making it easy to see patterns, trends, and fluctuations in your data.

## When to Use Line Charts

Line charts are ideal for:
- Showing trends over time
- Displaying continuous data
- Comparing multiple data series
- Tracking changes and patterns
- Analyzing time-series data

## Creating a Line Chart

To create a line chart in DataInsight AI:

1. Upload your CSV file with time-series or sequential data
2. Select your X-axis (typically time or sequence)
3. Select one or more Y-axis columns to plot
4. Choose "Line Chart" from the chart type selector
5. Customize your visualization

## Configuration Options

### Line Styles
- Solid, dashed, or dotted lines
- Line thickness
- Curved or straight connections
- Data point markers

### Multiple Series
- Compare multiple metrics on the same chart
- Different colors for each series
- Individual line styling
- Legend management

### Customization
- Axis ranges and scales
- Grid lines and reference lines
- Tooltips on hover
- Zoom and pan controls

## Best Practices

- Use consistent time intervals on the X-axis
- Avoid plotting too many lines (3-5 is optimal)
- Use distinct colors for different series
- Include data point markers for small datasets
- Add reference lines for important thresholds
- Label axes clearly with units

## Example Use Cases

- Stock price trends over time
- Website traffic analytics
- Temperature changes throughout the day
- Sales performance over months
- User growth metrics
- Product adoption rates
