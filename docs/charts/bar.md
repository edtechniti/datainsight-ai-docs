# Bar Charts

Bar charts are one of the most commonly used chart types for visualizing categorical data. They display data using rectangular bars with lengths proportional to the values they represent.

## When to Use Bar Charts

Bar charts are ideal for:
- Comparing values across different categories
- Showing rankings or comparisons
- Displaying discrete data
- Visualizing survey results
- Comparing performance metrics

## Creating a Bar Chart

To create a bar chart in DataInsight AI:

1. Upload your CSV file
2. Select the columns you want to visualize
3. Choose "Bar Chart" from the chart type selector
4. Configure your chart options

## Configuration Options

### Orientation
- **Vertical bars**: Default orientation, categories on X-axis
- **Horizontal bars**: Categories on Y-axis, useful for long category names

### Customization
- Colors and themes
- Bar spacing and width
- Axis labels and titles
- Legend position
- Grid lines

## Best Practices

- Use consistent colors for related data
- Keep bar spacing uniform
- Start the Y-axis at zero to avoid misleading visualizations
- Limit the number of categories for better readability
- Sort bars by value when showing rankings

## Example Use Cases

- Sales performance by product
- Revenue by region
- Survey responses by category
- Monthly expense comparisons
- Team performance metrics
