## Best Practices

### For Presentations

**Slides & PowerPoint**:
```
1. Export charts as PNG (high quality)
2. Use transparent background
3. Maintain consistent sizing
4. Export at 2x resolution for crisp display
```

### For Reports

**Documents & PDFs**:
```
1. Use PDF export for comprehensive reports
2. Include AI insights for context
3. Add data summary for reference
4. Use landscape orientation for wide charts
```

### For Print

**Printed Materials**:
```
1. Export at ultra quality (3x)
2. Use vector format if available
3. Test print in grayscale
4. Ensure text is readable at print size
```

### For Web

**Websites & Blogs**:
```
1. Export PNG at standard quality
2. Optimize file size
3. Use transparent background
4. Add alt text for accessibility
```

## Export Quality

### Image Quality Guidelines

| Use Case | Recommended Quality |
|----------|---------------------|
| Screen viewing | Standard (1x) |
| Presentations | High (2x) |
| Print materials | Ultra (3x) |
| Social media | Standard-High |
| Web embedding | Standard |

### File Size Considerations

**Typical file sizes**:
- PNG (standard): 50-200 KB
- PNG (high): 100-400 KB
- PDF (standard): 500 KB - 2 MB
- PDF (with data): 2-10 MB

**Reduce file size**:
- Use standard quality for screen viewing
- Exclude raw data from PDFs
- Compress PDFs after export
- Limit number of charts per PDF

## Troubleshooting

### Export Fails

**Problem**: Export button doesn't work

**Solutions**:
- Check browser allows downloads
- Disable popup blockers
- Try different browser
- Check available disk space

### Poor Image Quality

**Problem**: Exported images look blurry

**Solutions**:
- Increase export quality setting
- Use PNG instead of JPEG
- Export at higher resolution
- Check original chart render quality

### Large PDF Files

**Problem**: PDF file is too large

**Solutions**:
- Exclude raw data tables
- Reduce number of charts
- Use standard quality instead of high
- Compress PDF using external tool

### Wrong Colors

**Problem**: Colors don't match screen

**Solutions**:
- Check theme setting before export
- Use light theme for print
- Verify background color setting
- Test in PDF viewer

## API Export (Developer Feature)

For developers integrating DataInsight AI:

```typescript
// Export chart as PNG
const exportChart = async (chartId: string) => {
  const blob = await generateChartImage(chartId, {
    quality: 'high',
    background: 'transparent'
  });
  downloadBlob(blob, 'chart.png');
};

// Export report as PDF
const exportPDF = async (options: PDFOptions) => {
  const pdf = await generatePDFReport({
    includeInsights: true,
    includeCharts: true,
    includeData: false
  });
  downloadBlob(pdf, 'report.pdf');
};
```
