# What is DataInsight AI?

DataInsight AI is a modern, AI-powered web application that transforms your CSV data into beautiful, interactive visualizations with intelligent insights.

## Overview

DataInsight AI bridges the gap between raw data and meaningful insights by combining:

- **AI-Powered Analysis** using Claude AI
- **Interactive Visualizations** with multiple chart types
- **Natural Language Interface** for asking questions about your data
- **Privacy-First Architecture** - all processing happens in your browser

## Core Capabilities

### 1. Instant Data Analysis

Upload your CSV file and immediately receive:
- Summary of your dataset
- Key insights and patterns
- Anomaly detection
- Data quality assessment
- Actionable recommendations

### 2. Smart Visualizations

The AI analyzes your data structure and suggests the most appropriate chart types:
- Bar charts for categorical comparisons
- Line charts for trends over time
- Pie charts for distribution
- Scatter plots for correlations
- Area charts for cumulative values

### 3. Natural Language Queries

Ask questions about your data in plain English:
- "What's the average sales by region?"
- "Show me the trend over the last 6 months"
- "Which category has the highest values?"
- "Are there any outliers in this data?"

### 4. Export and Share

Share your insights with others:
- Export charts as PNG images
- Generate PDF reports with multiple visualizations
- Share via unique links (coming soon)

## Who Is It For?

DataInsight AI is perfect for:

- **Business Analysts** who need quick insights from data
- **Marketers** analyzing campaign performance
- **Researchers** exploring datasets
- **Students** working on data projects
- **Anyone** who wants to understand their data better

## Technology Stack

Built with modern web technologies:

- **React 19** - Fast, reactive user interface
- **TypeScript** - Type-safe code
- **Recharts** - Beautiful, responsive charts
- **Claude AI** - Intelligent data analysis
- **Vite** - Lightning-fast development and builds

## Privacy and Security

Your data privacy is our top priority:

- **No server uploads** - all CSV parsing happens in your browser
- **No data storage** - data exists only in your session
- **Secure API communication** - your Claude API key is stored locally
- **Open source** - inspect the code yourself
